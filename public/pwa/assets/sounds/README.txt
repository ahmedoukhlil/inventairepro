SONS REQUIS
===========

Vous devez créer les fichiers audio suivants dans ce dossier :

- beep-success.mp3 (son de succès, 0.5-1 seconde)
- beep-error.mp3 (son d'erreur, 0.5-1 seconde)

Format : MP3
Durée : 0.5 à 1 seconde
Volume : Modéré

Ressources gratuites :
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/
